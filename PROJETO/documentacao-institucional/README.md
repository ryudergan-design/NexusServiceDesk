# Documentacao Institucional

Esta pasta concentra a documentacao institucional e tecnica oficial do `Nexus ServiceDesk`.

## Artefatos
- `documentacao-institucional.html`: versao visual completa usada como base para o PDF, com galeria clicavel para ampliacao das imagens.
- `documentacao-institucional.pdf`: versao final pronta para apresentacao e compartilhamento.
- `assets/`: capturas reais das principais telas do sistema.

## Autor
- `Jeferson Conceicao`

## Objetivo
Registrar o produto de forma institucional, mostrando:
- visao executiva
- capacidades do sistema
- arquitetura
- stack, engine de IA e caminhos internos relevantes
- como rodar, testar e adaptar a camada de IA
- fluxos operacionais
- IA no atendimento
- estrategia de dados
- evidencias visuais reais

## Atualizacao
Sempre que houver mudanca relevante de produto, fluxo, arquitetura ou interface, revise:
- o HTML institucional
- o PDF gerado
- a `phase-9-documentacao-institucional`
