# Site

Esta pasta agrupa os pontos de referencia do frontend.

## Estrutura
- `docs/`
  Guias e catalogos do frontend.
- `../src/app`
  Rotas e paginas do Next.js.
- `../src/components`
  Componentes de interface.
- `../src/lib`
  Regras de negocio e integracoes usadas pelo frontend.

## Observacao
O codigo fonte continua em `src/` para preservar a estrutura nativa do Next.js e evitar quebra no runtime.
